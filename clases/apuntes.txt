Esta relacionado con objetos, creaicon mediante function y prototipos

La clase no es un objeto sino una plantilla para ahcer objetos en js
Las clases son una mejora sobre los prototitpos que se empezaron a usar a partie de la version 6 de js

ejemplo
class Coche{
    constructor(mar, mod){
        this.marca=mar
        this.modelo=mod;
    }
}
poner _ delante de los this

let miChoche = new Coche('mercedes', "yaris")
console.log( `${moCoche._marca}, ${miCoche._modelo}`)
